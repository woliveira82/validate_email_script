# validate_email_script

From help:
```bash
Usage:
    ./checkmail -f <filename> [OPTIONS]
Read a mailing text file called <filename> and create a new file called
EMAIL_SAFE_<filename> striping lines with wrong e-mail values.

Options:
    -f, --file <filename>       file to proccess
    -s, --sep <separator>       character that separate columns (default: |)
    -e, --email <email column>  column name with email field (default: email)
    -h, --help  display this help and exit
```
